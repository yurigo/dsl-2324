# TODOS

## Introducción

Servicio para hacer CRUD de todos.

- Ver todos los todos
- Añadir un todo.
- Eliminar un todo.
- Editar todo

Endpoints:

- /getAll
- /add
- /delete
- /update
- /do
- /undo

Cambios en la anatomía de un TODO:

```json
[
  {
    "id": 1,
    "text": "Limpiar la casa",
    "done": true
  },
  {
    "id": 2,
    "text": "Limpiar el lavabo",
    "done": false
  }
]
```
